## What it is

Demonstrative pronouns point to specific things or people. They come in two main types: **celui-series** (for pointing at specific nouns) and **ce/cette + noun** (for general demonstrations). The suffixes **-ci** and **-là** add proximity distance.

## Key forms

### Celui / Celle / Ceux / Celles

| Form | Use |
|------|-----|
| celui (masc sing) | this one / that one (masculine) |
| celle (fem sing) | this one / that one (feminine) |
| ceux (masc pl) | these ones / those ones (masculine) |
| celles (fem pl) | these ones / those ones (feminine) |

### With -ci / -là

| Form | Meaning |
|------|---------|
| celui-ci | this one (near) |
| celui-là | that one (far) |
| celle-ci | this one (feminine, near) |
| celle-là | that one (feminine, far) |
| ceux-ci | these ones (near) |
| ceux-là | those ones (far) |

### Ce / Cette + noun (not pronouns)

This is **not** a pronoun — it's a demonstrative adjective modifying a noun:

*Ce livre / Cette maison / Ces fleurs*

### Ceci / Cela / Ce

These are true pronouns that stand alone without referring to a specific noun:

| Form | Use |
|------|-----|
| ceci | this (near, formal/literary) |
| cela | that (far, most common) |
| ça | that (informal, colloquial) |
| ce (c') | subject of être |

*Ceci est intéressant.* — This is interesting.
*Cela m'inquiète.* — That worries me.
*Ça va ?* — How's it going? (literally: that goes)

## Usage rules

### Pointing at specific nouns — celui-series

When you point at a specific person or thing already mentioned or understood:

*J'aime ces deux voitures. Celle-ci est rouge, celle-là est bleue.*
—I like these two cars. This one is red, that one is blue.

*Le professeur et ceux qui écoutent* — The professor and those who are listening

### With relative clauses

Celui qui / que / dont + clause is common:

*Celui qui arrive tard* — The one who arrives late
*Celles que nous avons vues* — Those (feminine) that we saw

### Ceci / Cela (not pointing at specific nouns)

Use when referring to ideas, situations, or unspecified things:

*Ceci est important: mangez moins.* — This is important: eat less.
*Cela m'est égal.* — That's all the same to me.

### Ce + être

When **ce** is the subject of *être*, the verb agrees with the predicate:

*C'est moi.* — It's me. (c' = ce, predicate = moi)
*C'est eux.* — It's them.
*Ce sont les enfants.* — It's the children.

## Common pitfalls

### Ceci/cela vs celui-ci/là for people

**Never** use *ceci* or *cela* to point at people. Use the *celui* series:

*Celui qui est entré* ✓ — The one who entered
*Cela qui est entré* ✗

### Confusing demonstrative adjectives with pronouns

*Cette maison* = demonstrative adjective (modifies maison)
*Celle-ci* = demonstrative pronoun (stands alone)

The difference is clear from whether a noun follows.

### Ça vs Cela

In spoken French, **ça** replaces **cela** almost completely:

*Ça m'est égal.* ✓ (colloquial)
*Cela m'est égal.* ✓ (formal, written)

Both mean "that doesn't matter to me."

### Ce without être is not a demonstrative pronoun

*Ce livre* = "this/that book" — here "ce" is an adjective, not a pronoun.
