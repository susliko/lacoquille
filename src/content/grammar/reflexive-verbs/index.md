## What it is

A reflexive verb is a verb where the action returns to the subject. The subject performs an action on itself. Unlike English where "to wake up" doesn't use a reflexive pronoun, French uses **se réveiller** literally: "to wake oneself."

*Je me réveille.* — I wake myself up.

This isn't just a pronoun pattern — it's a full verb category that conjugates differently from regular verbs.

## The pronoun table

Reflexive pronouns match the subject. Unlike object pronouns, they don't have separate strong/weak forms:

| Subject | Reflexive Pronoun |
|---------|-------------------|
| je | me (m') |
| tu | te (t') |
| il/elle/on | se (s') |
| nous | nous |
| vous | vous |
| ils/elles | se (s') |

## Conjugation in simple tenses

Reflexive verbs conjugate like regular verbs, but always include the reflexive pronoun before the verb.

### Present

*se réveiller (to wake up)*

| Subject | Form |
|---------|------|
| je | me réveille |
| tu | te réveilles |
| il/elle/on | se réveille |
| nous | nous réveillons |
| vous | vous réveillez |
| ils/elles | se réveillent |

### Imparfait

*se réveiller in imparfait*

| Subject | Form |
|---------|------|
| je | me réveillais |
| tu | te réveillais |
| il/elle/on | se réveillait |
| nous | nous réveillions |
| vous | vous réveilliez |
| ils/elles | se réveillaient |

### Futur simple

*se réveiller in futur simple*

| Subject | Form |
|---------|------|
| je | me réveillerai |
| tu | te réveilleras |
| il/elle/on | se réveillera |
| nous | nous réveillerons |
| vous | vous réveillerez |
| ils/elles | se réveilleront |

## Conjugation in compound tenses

Reflexive verbs always use **être** as the auxiliary in compound tenses (passé composé, plus-que-parfait, etc.). This is the same auxiliary used by motion and state-of-being verbs (DR MRS VANDERTRAMP).

*Elle est partie.* — She left.
*Il s'est réveillé.* — He woke up.

### Agreement rules

The past participle agrees with the direct object when the reflexive pronoun is the direct object. This is the key rule:

**Direct reflexive** — the action is done to oneself:
*Elle s'est lavée.* — She washed (herself).
(se = direct object, feminine → agrees)

*Ils se sont regardés.* — They looked at each other.
(se = direct object, masculine plural → agrees)

**Indirect reflexive** — the reflexive pronoun is indirect, the direct object is something else:
*Elle s'est lavé les cheveux.* — She washed her hair.
(les cheveux = direct object, se = indirect → no agreement)

**Reciprocal** — each person does something to the other:
*Ils se sont regardés.* — They looked at each other. (each looked at the other)

## DR MRS VANDERTRAMP connection

Reflexive verbs use **être** as their auxiliary, same as the motion/state verbs:

| DR MRS VANDERTRAMP | Meaning | Example |
|--------------------|---------|---------|
| **R**ester | to stay | Je suis resté. |
| **M**ourir | to die | Il est mort. |
| **S**ortir | to go out | Nous sommes sortis. |
| **V**enir | to come | Vous êtes venu. |
| **A**lller | to go | Je suis allé. |
| **N**aître | to be born | Elle est née. |
| **D**escendre | to go down | Je suis descendu. |
| **E**ntrer | to enter | Nous sommes entrés. |
| **R**etourner | to return | Ils sont retournés. |
| **T**omber | to fall | Elle est tombée. |
| **R**entrer | to come back | Je suis rentré. |
| **A**rriver | to arrive | Vous êtes arrivé. |
| **M**onter | to go up | Je suis monté. |
| **P**artir | to leave | Il est parti. |

Reflexive verbs (se réveiller, se lever, s'habiller) follow the same pattern. **Always être, never avoir.**

## Passive se vs reflexive se

When se appears with an inanimate subject, the construction is passive, not reflexive:

*Ce livre se vend bien.* — This book sells well.
*Les portes se ferment à minuit.* — The doors close at midnight.

This is called the **onese** passive — se + third person verb creates a passive meaning. The subject isn't doing something to itself; the subject is being acted upon.

Compare:
- *Je me lave.* — I wash myself. (reflexive)
- *Ce livre se vend.* — The book sells. (passive)

## Impersonal se

Some expressions use se but have no true reflexive meaning:

*Il s'agit de…* — It's about…
*Il se peut que…* — It may be that…
*Il se passe quelque chose.* — Something is happening.

These are fixed expressions. Don't try to identify a reflexive relationship — just learn them as-is.
