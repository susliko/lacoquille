## What it is

In French, adjective position relative to the noun can be **obligatory**, **meaning-changing**, or **stylistic**. Unlike English (almost all adjectives precede), French has a default rule with notable exceptions.

## After the noun: the default

Most adjectives go **after** the noun. These describe objective, measurable qualities:

| Category | Examples |
|----------|----------|
| Nationality/Origin | *français, italien, chinois* |
| Shape | *rond, carré, ovale* |
| Color | *rouge, bleu, vert* |
| Material | *métallique, wooden, coton* |
| Purpose/Type | *médical, militaire, sportif* |

- *une maison **ancienne*** — an old house
- *une table **ronde*** — a round table
- *unvetement **bleu*** — a blue garment

## Before the noun: the BAGS group

A specific set — **BAGS** — usually precedes the noun:

| | Category | French examples |
|--|----------|----------------|
| **B**eauty | Aesthetic评价 | *beau, belle, laid, joli* |
| **A**ge | Age-related | *vieux, vieille, nouveau, nouvelle, jeune* |
| **G**oodness | Quality judgment | *bon, mauvais* |
| **S**ize | Physical size | *grand, petit* |

- *un **beau** jardin* — a beautiful garden
- *une **jeune** fille* — a young girl
- *un **bon** restaurant* — a good restaurant
- *un **petit** problème* — a small problem

Other pre-noun adjectives: *gentil*, *mauvais*, *dernier*, *seul*, *premier*

## Meaning changes with position

Some adjectives **change meaning** depending on placement:

| Adjective | Before noun | After noun |
|-----------|-------------|------------|
| *grand* | important, great | tall, big |
| *petit* | minor, petty | small, short |
| *cher* | dear | expensive |
| *mauvais* | wicked, bad | bad at (skill) |
| *propre* | own | clean |
| *dernier* | latest, final | last (in a series) |
| *seul* | only | alone |

**Examples:**

- *un **grand** homme* — a great man (Napoleon, Churchill)
- *un homme **grand*** — a tall man (physically)
- *mon **propre** livre* — my own book
- *un livre **propre*** — a clean book
- *la **dernière** semaine* — the final week (of a series)
- *la semaine **dernière*** — last week (time-wise)
- *une **propre** chambre* — one's own room
- *une chambre **propre*** — a clean room

## Multiple adjectives

When a noun has both pre-noun and post-noun adjectives:

- *Before + after:* **un **petit** jardin **carré** — a small square garden

When two adjectives modify the same noun:

**Both before:** Usually combined or reordered:
- *un **vieux petit** ami* — an old friend (both short)

**Both after:** Joined by *et*:
- *une robe **verte et bleue*** — a green and blue dress

**Mixed positions:**
- *un **petit** problème **grave*** — a small serious problem
- *une **jolie** femme **italienne*** — a pretty Italian woman

## The nuance rule

When position is free (not meaning-changing), both positions can work with subtle shifts:

| Position | Effect |
|----------|--------|
| **Pre-noun** | More subjective, emotional, emphatic |
| **Post-noun** | More objective, descriptive, literal |

- *un **petit** problème* — a small problem (concerning)
- *un problème **petit*** — a minor problem (less serious, almost trivial)

## Quick reference

**Before the noun (BAGS + opinion):**
Beauty · Age · Goodness/Badness · Size + gentil, mauvais, dernier, seul, premier

**After the noun (default):**
All other adjectives — nationality, color, shape, material, purpose, religion, etc.

**Meaning changes:**
Grand/petit (size vs quality) · Cher (dear vs expensive) · Propre (own vs clean) · Dernier/seul (sequence vs absolute)
