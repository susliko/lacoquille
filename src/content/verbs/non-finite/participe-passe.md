## What is the participe passé

The **participe passé** is the past participle — a non-finite verb form used to build compound tenses (like the passé composé) and to function as an adjective. Its form does not change, but it **agrees** with certain nouns depending on how it is used.

## How to form

| Verb type | Ending | Example |
|-----------|--------|---------|
| **-ER** | -é | *parler → parlé* |
| **-IR** | -i | *finir → fini* |
| **-RE** | -u | *vendre → vendu* |
| Irregular | various | *été, eu, fait, pris, mis, vu, dit, pris*… |

**Common irregular past participles:**

| Infinitive | Participe passé |
|------------|-----------------|
| *avoir* | eu |
| *être* | été |
| *faire* | fait |
| *prendre* | pris |
| *mettre* | mis |
| *voir* | vu |
| *dire* | dit |
| *courir* | couru |
| *vouloir* | voulu |
| *pouvoir* | pu |
| *devoir* | dû |
| *savoir* | su |

## Agreement rules

The participe passé behaves differently depending on whether it is used in a compound tense with **avoir**, with **être**, or as an **adjective**.

### With avoir — agreement with preceding direct object

The past participle agrees with a **direct object that precedes the verb**:

*J'ai lu le livre.* — No agreement (le livre follows the verb).
*Le livre que j'ai lu.* — Agreement! *Que* (= le livre) precedes *ai lu*.
*Les fleurs qu'il a achetées.* — Agreement — *qu'il* (= les fleurs) precedes *a achetées*.
*Je l'ai lue.* — Agreement — *l'* (= la livre, direct object) precedes *ai lu*.

**Key:** If the direct object follows the verb, no agreement. If it precedes, agreement.

### With être — passive voice (invariable)

When used with *être* to form a passive voice construction, the participe passé is **invariable**:

*Elle a été opérée.* — She was operated on. (passive, not a être verb of movement)
*Les victimes ont été secourues.* — The victims were rescued.

**Note:** This is **passive voice**, not the *être* auxiliary used for intransitive motion verbs. In passive constructions, the participe passé never agrees with the subject.

### With être — intransitive motion verbs (DR MRS P VANDERTRAMP)

Seventeen intransitive verbs use *être* as their auxiliary. With these, the participe passé agrees with the subject:

| Verb | Participe passé | Example |
|------|-----------------|---------|
| *aller* | allé | *Elle est allée à Paris.* |
| *venir* | venu | *Ils sont venus hier.* |
| *arriver* | arrivé | *Nous sommes arrivés tard.* |
| *partir* | parti | *Il est parti ce matin.* |
| *entrer* | entré | *Je suis entré第一家* |
| *sortir* | sorti | *Elle est sortie.* |
| *rester* | resté | *Nous sommes restés à la maison.* |
| *tomber* | tombé | *Elle est tombée.* |
| *venir* | revenu | *Ils sont revenus.* |
| *monter* | monté | *Il est monté.* |
| *descendre* | descendu | *Je suis descendu.* |
| *naître* | né | *Elle est née en 2000.* |
| *mourir* | mort | *Il est mort.* |
| *retourner* | retourné | *Nous sommes retournés.* |
| *passer* | passé | *Elle est passée.* |

These verbs all encode motion toward or away from a point, or a change of state. **Agreement is with the subject.**

### As an adjective (variable)

When the participe passé functions as an adjective, it agrees with the noun it modifies:

*une lettre écrite* — a letter (that is) written (feminine singular)
*des livres lus* — books (that have been) read (masculine plural)
*une table cassée* — a broken table

## The compound tense formation reminder

| Auxiliary | Used with |
|-----------|-----------|
| **avoir** | Most verbs |
| **être** | Reflexive verbs + DR MRS P VANDERTRAMP |

## Common pitfalls

- **Avoir agreement** is triggered only by a preceding direct object — this rarely happens in speech but is tested rigorously in writing.
- **Passive être** (*a été*) vs **intransitive être** (*est allé*) — only the latter triggers subject agreement.
- In compound tenses with *avoir*, the **past participle can never agree with the subject**.
